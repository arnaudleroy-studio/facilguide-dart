# facilguide

Multilingual guide utilities for [Dart](https://dart.dev).

Build tech guides with support for English, Spanish, French, Portuguese, and Italian.

## Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  facilguide: ^0.1.0
```

## Usage

```dart
import 'package:facilguide/facilguide.dart';

void main() {
  print('FacilGuide Dart utilities v$version');
  print('Platform: $baseUrl');
}
```

## Features

- Multilingual content utilities
- Support for 5 languages: EN, ES, FR, PT, IT
- Tech guide formatting helpers
